# SPEEDHOME Property Price Intelligence

Aplikasi web untuk mengumpulkan dan menganalisis data harga sewa properti dari SPEEDHOME.com.

## Tech Stack

- **Frontend:** Next.js 14 + React + TypeScript
- **Styling:** Tailwind CSS
- **HTTP Client:** Axios
- **Web Scraping:** Cheerio (Node.js)

## Cara Menjalankan

```bash
# Install dependencies
npm install

# Jalankan development server
npm run dev

# Build untuk production
npm run build

# Start production server
npm start
```

Aplikasi akan berjalan di `http://localhost:3000`

## Fitur

- Cari properti berdasarkan nama area
- Tampilan ringkasan harga (Average, Median, Modus)
- Tabel lengkap daftar unit
- Visualisasi data dengan chart
- Download data ke Excel/CSV
- Responsive untuk HP & Desktop
